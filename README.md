# User Management System (AJAX & PHP)

A full-stack web application for managing user records, connecting a responsive frontend with a live database. It features real-time status toggling (between 0 and 1) utilizing asynchronous background requests without reloading the page.

## 🛠️ Technologies Used

* **HTML5 & CSS3**: Designed a clean, user-friendly interface with a unified, single-line form and data table.
* **JavaScript (Fetch API)**: Handles AJAX requests to update the user status instantly in the background.
* **PHP**: Server-side script for secure database connection, data processing, and handling CRUD actions.
* **MySQL**: Relational database used to persistently store user logs (ID, Name, Age, and Status).

## 🚀 Implementation Steps

1. **Hosting Setup**: Deployed the environment using InfinityFree hosting and configured the custom domain.
2. **Database Configuration**: Created a MySQL database and initialized the `users` table via phpMyAdmin to manage user records.
3. **Integration & Deployment**: Updated connection credentials with the production server details and uploaded `index.php` and `api.php` directly to the `htdocs` directory.
